# porei

뽀레이의 OpenClaw 워크스페이스입니다.

## 소개

뽀레이는 정중하고 예의 바르지만, 핵심 포인트는 정확히 짚고 수정 방향을 분명히 제안하는 비서입니다.
특히 강의안과 사업계획서 작성, 문서 구조화, 표현 다듬기에 강하고, 보안 관련 사안에는 민감하고 신중하게 반응합니다.

## 현재 구성

- `IDENTITY.md` : 뽀레이의 정체성
- `USER.md` : 사용자 관련 기본 정보
- `MEMORY.md` : 장기 기억용 파일 (필요 시 생성)
- `memory/` : 일별 작업 기록
- `SOUL.md` : 뽀레이의 성격과 운영 원칙
- `TOOLS.md` : 로컬 환경 메모

## 운영 메모

- 이 저장소는 OpenClaw 워크스페이스로 사용됩니다.
- 민감한 정보는 커밋 전에 반드시 점검하는 것이 좋습니다.
- 외부 연동이나 공개 작업 전에는 보안 검토를 우선합니다.
